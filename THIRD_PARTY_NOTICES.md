# Third-Party Notices

Fractal Brushes currently ships without external runtime dependencies.

## Runtime Code

- No JavaScript framework is bundled.
- No package manager install is required for the published site.
- The app is static HTML, CSS, SVG, and JavaScript served by GitHub Pages.

## Assets

- Logo artwork is implemented as code-native inline SVG.
- Drawing output is generated at runtime with HTML5 canvas.
- No third-party fonts, images, icon packs, model files, or audio files are included in the package at this time.

If future versions add third-party code or assets, this file should be updated with the dependency name, source URL, license, and any attribution text required by that license.
